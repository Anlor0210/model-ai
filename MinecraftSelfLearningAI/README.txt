Cách chạy mô hình AI tự học chơi môi trường Minecraft đơn giản:

1. Cài đặt Python >= 3.8
2. Cài matplotlib nếu chưa có:
   pip install matplotlib

3. Chạy huấn luyện:
   python train.py

Mô hình sẽ tự học né zombie và ăn food trên bản đồ 7x7, bằng Q-learning.
Biểu đồ reward sẽ hiển thị quá trình học.
